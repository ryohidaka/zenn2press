Convert Zenn content format to VitePress.

Installation

You can install this library using npm:

npm install zenn2press

Usage

$ npm install -g zenn2press
$ zenn2press COMMAND
running command...
$ zenn2press (--version)
zenn2press/0.0.0 darwin-x64 node-v20.9.0
$ zenn2press --help [COMMAND]
USAGE
  $ zenn2press COMMAND
...
