## Installation

You can install this library using npm:

```shell
npm install zenn2press
```

# Usage

<!-- usage -->

```sh-session
$ npm install -g zenn2press
$ zenn2press COMMAND
running command...
$ zenn2press (--version)
zenn2press/0.0.0 linux-x64 node-v18.20.2
$ zenn2press --help [COMMAND]
USAGE
  $ zenn2press COMMAND
...
```

## `zenn2press`

Update installed plugins.

```
USAGE
  $ zenn2press  [-h] [-v]

FLAGS
  -h, --help     Show CLI help.
  -v, --verbose

DESCRIPTION
  Update installed plugins.
```
